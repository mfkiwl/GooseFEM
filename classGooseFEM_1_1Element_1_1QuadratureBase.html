Detailed Description

template<size_t ne, size_t nd, size_t td>
class GooseFEM::Element::QuadratureBase< ne, nd, td >

Base quadrature-class.

This class does not have a specific element-type in mind, it is used mostly internally to derive from such that common methods do not have to be reimplementation.

Template Parameters
neNumber of nodes per element.
ndNumber of dimensions for node vectors.
tdNumber of dimensions for integration point tensors.
